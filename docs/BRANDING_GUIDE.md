## üî§ Typography

### Font Family
**Primary**: Inter (or system fallback: -apple-system, BlinkMacSystemFont, Segoe UI, Roboto, sans-serif)

### Font Weights
- **Bold (700)**: Headings, emphasis
- **Semibold (600)**: Subheadings, labels
- **Regular (400)**: Body text, descriptions
- **Medium (500)**: Buttons, highlights

### Font Sizes

#### Desktop
- **H1**: 32px / 1.2 line-height
- **H2**: 28px / 1.3 line-height
- **H3**: 24px / 1.4 line-height
- **Body**: 16px / 1.5 line-height
- **Small**: 14px / 1.5 line-height
- **Tiny**: 12px / 1.4 line-height

#### Mobile
- **H1**: 28px / 1.2 line-height
- **H2**: 24px / 1.3 line-height
- **H3**: 20px / 1.4 line-height
- **Body**: 16px / 1.5 line-height
- **Small**: 14px / 1.5 line-height
- **Tiny**: 12px / 1.4 line-height

### Typography Rules
- Use consistent line-height for readability
- Maintain hierarchy with size and weight
- Limit to 2-3 font sizes per page
- Ensure minimum 16px for body text on mobile

## üé§ Tone of Voice

### Brand Voice Characteristics
- **Conversational**: Friendly, not corporate
- **Clear**: Simple, jargon-free language
- **Encouraging**: Positive, motivating tone
- **Helpful**: Supportive, solution-focused
- **Authentic**: Genuine, not overly formal

### Writing Guidelines

#### Do's ‚úÖ
- Use "you" and "we" to create connection
- Be specific and concrete
- Use active voice
- Keep sentences short and simple
- Use contractions (it's, we're, don't)
- Be encouraging and positive

#### Don'ts ‚ùå
- Don't use corporate jargon
- Don't be overly formal or stiff
- Don't use passive voice
- Don't use complex terminology
- Don't be negative or discouraging
- Don't make assumptions about users

### Example Copy

**Good**: "Join a tournament near you this weekend"  
**Bad**: "Participate in a sporting event in your geographical vicinity"

**Good**: "You're all set! See you at the tournament"  
**Bad**: "Registration has been successfully processed"
